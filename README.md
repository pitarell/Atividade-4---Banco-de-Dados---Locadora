# Mini-mundo Locadora de Veículos – Atividade 4 (DML)

Este repositório contém os scripts SQL de criação e manipulação de dados
do banco de dados da locadora de veículos.

## Arquivos

- `sql/01_inserts.sql`  
  Inserção de dados nas tabelas CLIENTE, VEICULO e LOCACAO.

- `sql/02_selects.sql`  
  Consultas com `SELECT`, `WHERE`, `ORDER BY`, `LIMIT`, `JOIN` e agregação.

- `sql/03_updates_deletes.sql`  
  Comandos `UPDATE` e `DELETE` com condições, respeitando integridade.

## Como usar

1. Abrir o arquivo `locadora.db` com o SQLTools (SQLite) no VS Code.
2. Executar primeiro o script de criação das tabelas (DDL).
3. Executar `sql/01_inserts.sql` para povoar as tabelas.
4. Executar `sql/02_selects.sql` para testar consultas.
5. Executar `sql/03_updates_deletes.sql` para atualizações e exclusões.

O projeto utiliza SQLite integrado ao VS Code via extensão SQLTools.
